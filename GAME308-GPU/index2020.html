Assignment 1:

For first few sessions I have tried to take you on a tour of the Khronos sponsored tutorial of "Hello Triangle."
The fairly high level of C++ sophistication and its 900+ lines of code makes this a very daunting challenge.
There is no way to know what calls to make and in what order without the assistance of the group of engineers that created
the modern GPUs and this language. This is not easy stuff but this is were we are at in the high performance graphics world.
Welcome the complexity of Vulkan generated by the demands of of the gaming community, that's you all you people!

What you need to do is port the Vulkan tutorial from GLFW to SDL and get a triangle up. Let's make the due date next week.


If you have any technical savvy, you would have seen that the author of this website has posted
a github depository of the total source code of this first demo (and much more).

The idea, my dear souls, is that you need to appreciate what this new system looks like and what needs to be set up.
This technology will change constantly over your careers, that I can guarantee! What I'm trying to help you with the skills
to learn new things that will always be happening in our field.
